## Getting Started

First, run the development server:

```bash
npm run dev
```
## Pre-Requisites:

1. Install nodejs 20.11.0 or above `https://nodejs.org/en/download/`
2. Install yarn 1.22.21 or above `yarn install`
 
---------------------------------------------------------------------------------------------------------
## Technologies Used:

1. Next.js
2. TypeScript
3. Jest
4. Yarn
5. Material-UI (MUI) with SCSS
